# ソーラー発電

- 1チャンクをベースにそれっぽい比にする
  - ソーラーで埋め立てる場合、1チャンク当たり2マス幅(蓄電池が入る幅)が余る。
    - あまり幅が2と3の最小公倍数になる3チャンクx3チャンクで構成するときれいになるかも
- そも電池のほうが多いので改善したほうがいい
- マップ上だと割ときれいに見える

## 1チャンク

- 構成
  | ソーラー | 蓄電池 | 広域電柱 | ロボステ | レーダー | 比率(ソーラー/蓄電池) |
  | -------- | ------ | -------- | -------- | -------- | --------------------- |
  | 72       | 86     | 4        | 1        | 0        | 0.837                 |
